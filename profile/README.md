# Golem Linux
Linux distro based upon Void Linux with BusyBox as user land

## Software
List of packages for specific usecases by default

| User case | Software              |
|-----------|-----------------------|
| Bootloader| limine                |
| User land | bysbox-huge           |
| Kernel    | linux-mainline        |
| Network   | iwd openresolvd dhcpcd|
| Shell     | yash                  |
| Init      | runit                 |
| Libc      | glibc                 |
